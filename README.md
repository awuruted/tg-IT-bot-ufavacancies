# tg-IT-bot-ufavacancies
Парсер собирает актуальные вакансии сразу с четырёх площадок: zarplata.ru, hh.ru, SuperJob и государственного портала Trudvsem.ru. Каждый источник подключён через официальный API, что обеспечивает стабильность и корректность получаемых данных. Обновление происходит раз в час без участия пользователя — система сама следит за актуальностью базы
